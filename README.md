# EcoScan Project

## Description:
EcoScan is a web application that helps users make environmentally responsible choices by scanning products, analyzing their ecological impact, and suggesting sustainable alternatives.

---

## Features
- Product scanning (barcode)
- Eco impact score
- Sustainable alternatives
- User challenges and progress tracking

---

## Technologies:
- PHP / Laravel (or whatever she uses)
- MySQL (phpMyAdmin)
- HTML/CSS/JS
